<img width="600" src="https://github.com/IvanShavliuga/vue-chat/blob/master/intro.png">

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
